# Cinema Frontend

Next.js application for Cinema ticket booking system.

## Getting Started

Install dependencies:
```bash
npm install
```

Run development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Environment Variables

Create `.env.local`:
```
NEXT_PUBLIC_API_URL=http://localhost:8080/api
```

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Axios
- React Query
